My First Octopress Blog

Jan 12th, 2014 | Comments

突然想利用Blog记录自己工作上的一些心得以及作为一个FAQ，然后从同事那里得知了这么一个平台，于是开始试着折腾（windows下搭建这个真的挺麻烦的）。 先记录下自己搭建的过程吧。

参考文章：
http://www.cnblogs.com/oec2003/archive/2013/05/27/3100896.html
http://www.cnblogs.com/oec2003/archive/2013/05/31/3109577.html

一、下载相应的软件

本人用的是Windows 8.1

GitHub for windows: https://github-windows.s3.amazonaws.com/GitHubSetup.exe
Ruby: http://rubyforge.org/frs/download.php/76798/rubyinstaller-1.9.3-p392.exe
DevKit: http://cloud.github.com/downloads/oneclick/rubyinstaller/DevKit-tdm-32-4.5.2-20111229-1559-sfx.exe
Python：http://www.python.org/ftp/python/2.7.5/python-2.7.5.msi(貌似是用来高亮语法用的，可以暂时不装)
Octopress: git://github.com/imathis/octopress.git(先不用下)

二、安装软件

GitHub
直接安装，也没什么选的。

Ruby
要在DevKit之前安装。安装的时候有个步骤有选项，我只选了前两个

DevKit
解压，路径中没有中文，然后在命令行中进入到DevKit的目录下安装
例如：

我没装Python……所以请参照参考文章

Octopress
我在E盘专门建立了个GitHub文件夹放blog的工程文件

安装octopress的依赖项。我没有更换source。

安装octopress默认主题

rake install

然后可以预览了

rake preview

在浏览器中打开local:4000就可以看到blog的原始页面

三、使用中文

如果要在blog中使用中文，需要做些设置。

1.添加系统环境变量

2.含有中文的文档要保存成UTF-8无BOM编码格式。
win的记事本无法做到，推荐使用UltraEdit
http://down.05sun.com/crack/UltraEdit_SC.rar

3.在ruby安装路径下找到文件convertible.rb
例如：
E:\Ruby193\lib\ruby\gems\1.9.1\gems\jekyll-0.12.0\lib\jekyll\convertible.rb
把self.content这行修改成
self.content = File.read(File.join(base, name), :encoding => ‘utf-8’)
